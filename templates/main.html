Corechat

Welcome to my very project.This is my First project using Flask and python for backend and html css and javascript for frontend.

Expected Features

  • Python programming using object-oriented concepts wherever possible
  • Basic documentation for requirements (using use-cases)
  • Basic design document
  • Web server and client implementation using REST API
  • Works on a single machine
  • Text interface
  • Users can sign up, log in, log out, list existing users, see messages, and send new messages

Non-Requirements

  • No GUI interface
  • No elaborate password-saving techniques; simply storing them in a file is sufficient

Additional Project Extensions (Optional)

  • Storing data in a database instead of a file
  • Implementing HTTPS
  • Group messaging
  • Test cases (using pytest)

Learning Objectives

  • Documentation: requirements, design document, test cases
  • Web server basics: HTTP, port, connection, etc.
  • REST APIs
